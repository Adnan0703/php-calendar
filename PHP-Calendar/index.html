PHP-Calendar
Intent
I was searching for a decent calendar which my group at school could use to keep track of events, etc. We were previously using localendar, which I didn't like and it had some problems. I found CST-Calendar which did most of what I wanted, but was rather ugly and missed some features others in the group wanted. So, I changed CST-Calendar drastically, and that project seems to have stopped work, so I'm releasing this independently. If you'd like to contribute of have any question or suggestions, contact me at sproctor@ccs.neu.edu
